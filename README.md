# CDS-CONSULTORIA

Corporate website for CDS Consultoría Informática, an IT consulting firm.

## Tech Stack
- HTML5
- CSS3 (embedded)
- JavaScript (inline)
- Google Fonts (Rajdhani, Mulish)

## Setup
Open index.html in a web browser to view the site.

## Files
- index.html: Main landing page
- ecosistema_cds_nexo.html: Ecosystem page
- Firma-cds.html: Company signature page
